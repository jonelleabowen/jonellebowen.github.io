Introduction¶

Data Science is an increasingly popular discipline which is focused on using methods and scientific approaches to uncover useful insights from a given pool of data. Data comes in all forms, many of which are unideal, so in this tutorial, we'll take you through the full data science pipeline which involves gathering and cleaning data, doing some exploratory data analysis which will allow us to come up with some curious hypotheses, and testing out hypotheses using machine learning.

Tutorial Goals¶

Over the past few years, we've seen many instances on the news or social media of civilians being shot and killed by police officers in the line of duty. Having found an interesting dataset from https://www.kaggle.com/washingtonpost/police-shootings we're going to apply our data science methods to see if we can uncover any interesting patterns in these instances of police shootings. We think this is an especially interesting topic to analyze given the increased emphasis on police surveillance and accountability. The data was collected by the Washington Post and comprises every fatal shooting by police officers since January 1, 2015 to December 13, 2017

    Data Tidying¶

    We want our data to be uniform and standardized for processing because data often starts off messy. The first thing we will do is create a standard way of representing any missing information by replacing it with 'Unknown'.

    For further analysis on any potential relationship between the fatal shootings and the dates, we will convert each date string to a datetime object. DateTime is a standard way of representing dates, which is provided through a Python library. For further information, refer to: https://docs.python.org/2/library/datetime.html

    Machine Learning¶

    Machine Learning will allows us to perform some predictive analysis on our data. We are interested in predicting the the number of shootings for next year for any given month when both either body cameras are in use and are not. This prediction will allow us make statistical statements about the trend of shootings for the future such as if it will decrease or increase.

    We decided to separte the entire data set into two for better predictions. One set contains the data on shooting where body cameras were in use while the other contains the data where body cameras were no used.

    We do this by separating the data into tranning data and testing data. The training data is used to build a model that will be used to predict unobserved values for month and year (e.g., the future). On the other hand, the test data is used to check if our model is appropriate and returns accurate results.

    The function below implements the model and the model testing explained above.

# function used to test our model
# Returns an array that 
def kFold_other(K, X, y):
    
    regression = linear_model.LinearRegression()
    avg_train_error = 0
    kf = KFold(n_splits=K)
    errors = []
    
    for train, test in kf.split(X):

        x_train, y_train = X[train], y[train]
        x_test, y_test = X[test], y[test]

        regression.fit(x_train, y_train)

        predict = regression.predict(x_test)
        error = (predict - y_test) ** 2
        avg_error = np.mean(error) / K

        errors.append(avg_error)
    
    return errors

    Conclusion¶

    We reject our hyphothesis that in July 2018 there will be less shootings when body cameras are in use. However, after predicting the number of shootings for that date for both in use and not in use cases, we saw virtually no difference in predicted number of shootings.

    This could be explained by having a poor relationship between time and the number of shootings. We could further explore relationships between race and time or age to if there is a correlation.

    In addition, because body cameras are still relativally new, and may not have enough observations to give us the full picture.

    Lastly, currently there is no uniform regulations regarding body camera usage. For example, the officers might be allowed to turn the camera off when they desire.
